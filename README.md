# Lab 4 — Spring Boot + Vue

## 📌 Описание проекта
Данный проект представляет собой лабораторную работу, выполненную с использованием технологий **Spring Boot** (backend) и **Vue.js** (frontend).

Приложение реализует клиент-серверную архитектуру, где:
- backend предоставляет REST API
- frontend взаимодействует с сервером и отображает данные пользователю

## 🛠️ Используемые технологии
- Java
- Spring Boot
- Maven
- Vue.js
- Node.js / npm
- REST API

## 📂 Структура проекта
Проект разделён на две основные части:

- `backend` — серверная часть (Spring Boot)
- `frontend` — клиентская часть (Vue.js)

Подобное разделение является стандартным для fullstack-приложений :contentReference[oaicite:0]{index=0}

## ⚙️ Требования
Перед запуском убедитесь, что у вас установлены:
- Java 11+

- Maven
